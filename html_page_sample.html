Early Tech History

Fun Fact: The first computer bug was literally a bug! In 1947, engineers found a moth stuck in a relay of the Harvard Mark II computer, coining the term "debugging."

Fun Fact: The Qwerty keyboard layout was designed to slow typists down to prevent mechanical typewriters from jamming, not for efficiency.

Internet Wonders

Fun Fact: The first website (info.cern.ch) went live in 1991 and is still online today. It explained what the World Wide Web was.

Fun Fact: About 90% of the world's data was generated in just the last two years. We now create more data in 10 minutes than all of humanity did until 2003!

Tech Giants Trivia

Fun Fact: Google's first storage was made from LEGO bricks. They needed an expandable, cheap way to mount 10 4GB hard drives in 1996.

Fun Fact: Amazon started as an online bookstore in Jeff Bezos' garage. The first book sold was "Fluid Concepts and Creative Analogies" in 1995.

Programming Peculiarities

Fun Fact: There are programming languages named after sandwiches (Python), TV shows (Ruby), and even a programming language called "Chef" where programs look like cooking recipes!

Fun Fact: The first computer programmer was a woman - Ada Lovelace wrote the first algorithm for Charles Babbage's Analytical Engine in the 1840s.
